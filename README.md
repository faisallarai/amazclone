# Amazclone

# Technologies

- ReactJS + NextJS 13
- TailwindCSS
- Paystack (Accept Payment + Webhook)
- NextAuth
- Next Connect
- Google Login

## How to use

```bash
git clone https://github.com/faisallarai/amazclone.git
cd amazclone
yarn install
yarn dev
```
